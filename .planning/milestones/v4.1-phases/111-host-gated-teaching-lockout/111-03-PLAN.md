Client-side defense-in-depth for the host teaching lockout (LOCK-01, criterion 2): when the broadcast flag is set, GameShell/ControlsMenu hides the Get-a-hint, Watch-AI-demo, Show-move-quality, and Start-tutorial affordances. The flag is read primarily from broadcast state (single source of truth — criterion 4), with the platform `init` message consumed for first-render before the first broadcast (criterion 1).

Purpose: Hide what the player cannot use. The server (Plans 01/02) is the real enforcement; this prevents the affordances from even appearing.
Output: `teachingDisabled` consumed from init + a broadcast-preferred computed + composed AND-gate into the existing showHint/hasTutorial gates, passed as a `teachingDisabled` prop into ControlsMenu which suppresses the Teaching and Tutorial render groups.
</objective>
