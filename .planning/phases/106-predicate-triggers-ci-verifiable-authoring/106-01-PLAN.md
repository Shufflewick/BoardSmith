## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-106-01 | Tampering | author predicate mutating game state during evaluation | mitigate | TutorialGateContext.game is documented read-only; evaluation occurs in a read-only hot path (mirrors action conditions); no mutation API is handed to predicates |
| T-106-02 | Denial of Service | autoAdvanceTutorial infinite loop (predicate always fires) | mitigate | pump bounded by def.steps.length and terminated by nextProgress completion — forward-only, cannot revisit a step |
| T-106-03 | Tampering | predicate throws mid-evaluation | mitigate | evaluateConditionWithTrace try/catch records the failing label and treats it as `passed=false`; a throwing advanceWhen simply does not fire |
</threat_model>
