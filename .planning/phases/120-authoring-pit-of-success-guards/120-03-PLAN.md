<objective>
PIT-02: During the FIRST `startFlow()` traversal, record every element class queried through the `ElementCollection._finder()` choke point, then diff the recorded set against the class registry. Throw, naming the class and steering to `game.registerElements([Class])`, on the first class that was queried but never registered. Post-start / async queries are out of scope (documented limitation). Recording is disabled after the first traversal so it imposes no ongoing overhead.

Purpose: A class queried via `game.all(Foo)` but never registered (typo or dead class reference) currently misbehaves silently. This makes it a loud, named, actionable start-time error.
Output: A recording hook on `_finder()`, a first-traversal recording window in `startFlow()`, and PIT-02 unit tests.
</objective>
