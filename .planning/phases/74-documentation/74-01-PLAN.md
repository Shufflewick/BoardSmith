- `flyCard(options)` -> `fly(config)` where config uses `elementData`/`elementSize` instead of `cardData`/`cardSize`
- `flyCards(options)` -> `flyMultiple(configs)` with same property renames
- `FlyCardOptions` -> `FlyConfig` with property renames:
